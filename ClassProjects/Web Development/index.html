53: CSS Sizing

Styling Instructions:

  1. Set the h1 font size to 90px
    • setting it to 90px will not result in a dynamic fontsize
    • In order to create a dynamic fontsize, we use a %age. Default 100% = 16px.
    • We can also use the 'em' value for the fontsize property
      • In this case the 'em' is the pheonetic representation of the capital letter 'M'. So 1em = the width of 1 letter m. 1 em sized letter M = 16 px.
  2. Because most people now use the zoom function on their browser instead of setting a defualt font size in the browser's settings this distinction between dynamic and static font sizes is less important than it used to be.
  3. What is important however is that font sizes are inherited
    • So if a parent has a font size of 2em, and a child has a font size of 5.5em, the child will display a font size of 7.5em as the inherited size is added to the declared size.
      • You can get around this by using the 'rem'
      • it stands for 'root em'
      • this will ignore any upstream font sizes and not inherit additive sizes from parents.
    • This functions exactly the same way with percentages.
    • This is not true with sizes declared in px becuase they are non-dynamic sizes
